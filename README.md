# Calculator-Kivy
Aplicação de calculadora desenvolvida em Python utilizando o framework Kivy para criação de interfaces gráficas multiplataforma.

O projeto permite realizar operações matemáticas básicas por meio de uma interface intuitiva e responsiva, explorando conceitos de desenvolvimento de aplicações desktop com Python e Kivy.

Funcionalidades
Soma
Subtração
Multiplicação
Divisão
Limpeza da tela
Interface gráfica interativa
Tecnologias Utilizadas
Python
Kivy
Objetivo

Este projeto foi desenvolvido com o objetivo de praticar programação em Python, desenvolvimento de interfaces gráficas e utilização do framework Kivy, consolidando conhecimentos em lógica de programação e criação de aplicações desktop.
